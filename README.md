# code-picasso

Obfuscates your JavaScript file into an unreadable ASCII art skull that actually executes perfectly.

## 🚀 Usage

No installation required! Just run it directly using `npx`:

```bash
npx code-picasso
```

Or, install it globally:

```bash
npm install -g code-picasso
```
